# KoinxAssignment



# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)

# Environment vars
This project uses the following environment variables:
fill in the corresponding field inside the .env file.
| Name                          | Description                         |
| ----------------------------- | ------------------------------------|
|MONGO_URL           | URL of deployed database on mongodb atlas      |
|API_KEY             | API key of etherscan                           |

# Getting started
- Clone the repository
```
git clone git@github.com:tech-akash/KoinxAssignment.git
```

- Install dependencies
```
npm install
```
- Build and run the project
```
npm run start
```
Navigate to `http://localhost:8080

- API Document endpoints

  All Transactions Endpoint : http://localhost:8080/transaction/all-transactions/:userAddress

  Get Balance  Endpoint : http://localhost:8080/transaction/balance/:userAddress 
`
